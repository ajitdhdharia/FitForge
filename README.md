
# FitForge

## Getting Started

### Dependencies

- React
- React Router
- Mongoose
- MongoDB

### Installing
- Clone the repo:
  ```
  git clone https://github.com/ajitdhdharia/FitForge.git
  ```

### Executing program

- Frontend
  ```
  cd frontend
  npm install
  npm start
  ```
- Backend
  ```
  cd backend
  npm install
  npm run dev
  ```

## Built With

- MongoDB - The database used
- Express.js - The web framework used
- React - The front-end framework used
- Node.js - The back-end runtime environment
